# Fast Bookmark UI 现代化与样式重构计划

本计划旨在通过应用 "Modern Clean + Glassmorphism" (现代极简 + 毛玻璃) 设计语言，全面提升插件的视觉质感和交互体验。

## 1. 设计理念 (Design Philosophy)
- **Glassmorphism (毛玻璃)**: 侧边栏背景采用半透明磨砂效果，提升现代感，使插件与网页内容更有机地结合。
- **Card-like Interactivity (卡片式交互)**: 书签列表项在悬停时呈现圆角卡片形态，配合微妙的位移，提供明确的触觉反馈。
- **Refined Typography & Spacing (精致排版)**: 增加留白，优化行高和字重，提升信息阅读效率。

## 2. 具体优化内容

### 2.1 核心样式系统 (`:host` 变量)
- **背景**: 将纯色背景改为带透明度的背景色 (e.g., `rgba(255, 255, 255, 0.9)` )，配合 `backdrop-filter: blur(16px)`。
- **阴影**: 引入更柔和的多层阴影 (`box-shadow`)，替代单一的黑色阴影。
- **圆角**: 定义统一的圆角变量 (`--radius-md: 12px`, `--radius-sm: 8px`)。
- **动效**: 统一定义过渡曲线 (`cubic-bezier`)，使交互更丝滑。

### 2.2 组件级优化
- **侧边栏容器 (`#fast-bookmark-modal`)**:
    - 应用毛玻璃背景。
    - 添加 1px 的内描边 (`border`) 以增加精致感。
- **搜索框 (`#fast-bookmark-search-input-wrapper`)**:
    - 增加高度，使用更大的圆角。
    - 聚焦时添加主色调的光晕 (`ring`)。
    - 图标颜色更柔和。
- **书签列表 (`.fast-bookmark-result-item`)**:
    - 增加左右 Margin，使列表项悬浮于面板中间，而不是贴边。
    - Hover 状态添加 `transform: translateX(4px)` 位移。
    - 选中状态使用更明显的左侧指示条或背景色。
- **滚动条**:
    - 样式窄化，默认隐藏或半透明，悬停时加深。
- **设置面板**:
    - 优化滑块 (Range Slider)、复选框和下拉菜单的样式，使其符合整体设计语言。

## 3. 技术实施路径
主要修改集中在 `content.js` 的 `updateStyles` 函数中。

1.  **重构 CSS 变量**: 更新颜色、圆角、阴影等基础变量。
2.  **重写容器样式**: 为 `#fast-bookmark-modal` 添加 `backdrop-filter`。
3.  **优化列表渲染**: 调整 `renderNode` 和 `doRender` 生成的 HTML 结构（如果需要）或仅通过 CSS 调整布局（优先 CSS）。
4.  **微调交互**: 调整 `:hover`, `:focus` 状态的 CSS 规则。

## 4. 预期效果
插件将摆脱“原生 HTML”的粗糙感，呈现出类似 macOS 原生应用或现代 Web App 的精致外观。
