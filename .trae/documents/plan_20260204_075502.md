## 任务说明
在设置面板中添加“高亮色”配置项，允许用户分别设置白天模式和黑夜模式下的高亮颜色。

## 实施方案

### 1. 修改数据结构
- 在 [content.js](file:///Users/pandegong/my-project/fast-bookmark/content.js) 的 `settings` 对象中添加两个新字段：
  - `highlightColorLight`: 默认值 `#3730a3` (Darker Indigo)
  - `highlightColorDark`: 默认值 `#3730a3` (Same default)
- 更新 `loadSettings` 函数以支持这两个新字段的读取和默认值设置。

### 2. 更新样式逻辑
- 修改 `updateStyles` 函数：
  - 根据当前的 `isDark` 状态，选择对应的 `settings.highlightColorDark` 或 `settings.highlightColorLight`。
  - 将选定的颜色赋值给 `--primary-color` CSS 变量（或直接用于 `--accent-color`，因为原代码中 `--accent-color: var(--primary-color)`）。
  - 注意：原代码中 `--primary-color` 在黑夜模式下也是 `#3730a3`，我们需要确保用户配置能覆盖这个默认值。

### 3. 修改设置对话框 UI
- 在设置模态框中添加两个新的 `settings-row`：
  - **白天模式高亮色**：包含一个标签和一个颜色选择器 (`<input type="color">`)。
  - **黑夜模式高亮色**：同上。
- 在打开设置对话框时，初始化这两个颜色选择器的值为当前的配置值。

### 4. 保存设置
- 在保存按钮的点击事件中，获取两个颜色选择器的值，并更新 `settings` 对象和 `chrome.storage.sync`。
- 保存后调用 `updateStyles()` 以实时应用新的颜色。

### 5. 国际化支持
- 在 `_locales/en/messages.json` 和 `_locales/zh_CN/messages.json` 中添加相关翻译：
  - `highlightColorLightLabel`: "Highlight Color (Light Mode)" / "高亮色（明亮模式）"
  - `highlightColorDarkLabel`: "Highlight Color (Dark Mode)" / "高亮色（暗黑模式）"
