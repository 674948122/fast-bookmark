I will modify the `formatRelativeTime` function to display the absolute date and time (e.g., "YYYY-MM-DD HH:mm") instead of relative time (e.g., "Just now").

- **Implementation**:
  - Use `new Date(timestamp)`.
  - Format to a string using `toLocaleString()` or a custom format like `YYYY-MM-DD HH:mm`.
  - Given the user's preference for "concrete time" (具体时间), a compact `YYYY-MM-DD HH:mm` or standard locale string is best.
  - Considering internationalization, I should check the current language setting (`settings.language`) or just use the browser's default locale.
  - **Proposed Format**: `YYYY-MM-DD HH:mm` is universally understood and compact for the UI.
  - **Refinement**:
    ```javascript
    function formatTime(timestamp) {
        if (!timestamp) return "";
        const date = new Date(timestamp);
        // Zero-pad helper
        const pad = (n) => n.toString().padStart(2, '0');
        return `${date.getFullYear()}-${pad(date.getMonth() + 1)}-${pad(date.getDate())} ${pad(date.getHours())}:${pad(date.getMinutes())}`;
    }
    ```
