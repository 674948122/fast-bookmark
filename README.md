# Fast Bookmark

Chrome 原生书签栏会挤压网页空间，尤其是在笔记本等小屏幕设备上体验不佳。Fast Bookmark 是一款 Chrome 扩展程序，通过快捷键唤起一个悬浮在网页之上的搜索框，让你无需显示书签栏也能快速找到并打开书签。

## ✨ 特性

- **悬浮设计**：完全不占用网页空间，用完即走。
- **模糊搜索**：基于 Fuse.js 的高性能模糊匹配，支持标题和 URL 搜索。
- **快捷键支持**：全键盘操作，手不离键盘即可完成打开书签。
- **样式隔离**：使用 Shadow DOM 技术，不受原网页 CSS 影响。
- **暗色模式适配**：现代化的 UI 设计，清晰易读。

## 🚀 安装指南

由于本插件尚未发布到 Chrome 商店，你需要通过“开发者模式”进行安装：

1. 下载本项目代码到本地。
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3. 打开右上角的 **“开发者模式”** 开关。
4. 点击左上角的 **“加载已解压的扩展程序”**。
5. 选择本项目所在的文件夹（包含 `manifest.json` 的目录）。

## 📖 使用指南

### 唤起/关闭
- **点击图标**: 点击浏览器工具栏上的 **Fast Bookmark (悬浮书签)** 图标即可唤起或关闭搜索框。
- **Esc 键**: 在搜索框打开时，按 `Esc` 键或点击搜索框外部区域也可关闭。

### 搜索与导航
- 输入关键词进行实时模糊搜索。
- 使用 `↑` `↓` 方向键在结果列表中移动。
- 鼠标悬停也可选择。

### 打开书签
- **鼠标点击**: 直接点击书签项，将在 **新标签页** 中打开。
- **当前标签页打开**: 选中后按 `Enter`。
- **新标签页打开**: 选中后按 `Ctrl + Enter` (Windows/Linux) 或 `Command + Enter` (macOS)。
- **快捷复制**: 按住 `Alt` 键并点击书签项，可直接复制 URL。

## 🛠️ 技术栈

- Manifest V3
- Vanilla JS (无框架依赖)
- Fuse.js (模糊搜索)
- Shadow DOM (样式隔离)
